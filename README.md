Personal Portfolio Website

This project is a responsive personal portfolio website developed as part of a Full Stack Web Development Internship. The website showcases my skills, projects, and professional profile in a clean and modern design.

Features

- Interactive portfolio sections (About, Skills, Projects, Resume)
- Downloadable resume (CV)
- Fully functional contact form with email notifications using EmailJS
- Responsive and user-friendly interface
- SEO-friendly structure for better visibility

Technologies Used

- HTML
- CSS
- JavaScript
- EmailJS (for contact form functionality)

Purpose

The purpose of this project is to create a professional online presence and demonstrate my front-end development skills.

Outcome

This project helped me understand website structure, UI design, and integration of real-world features like contact forms and deployment.
